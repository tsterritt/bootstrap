Text Input Types

Bootstrap supports all HTML5 input types - email, text, tel, url, number, range, search, password, date and time, color
The input types don't all offer styling differences, however may trigger validations or offer attributes to contorl behavior

form-range class adds some styling
form-control-color class constrains the input to a square
