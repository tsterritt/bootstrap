Checkbox

form-check class on the outer div adds indentation, form-check-input on the input adds styling on the checkbox


Add checked attribute to input element to show it as checked by default

Indeterminate is set using javascript

Disabled

Add disabled attribute to input element to show as disabled. Can use disabled and checked attributes on same element

Radio

Use the name attriute on the radio inputs to link them together so options are mutually exclusive

Switch

Use the form-switch class on the containing div to make the checkboxes a switch instead of a checkbox

Can use switches in radio groups using the radio input type

Inline

Use the form-check-inline class on the outer form-check divs to make checkbox elements inline

No Labels

Leave off the label elements to create checks with no labels

Checks Styled as Buttons

Use the btn-check class on the input element and the btn classes on the label to create button styled checks. Use outline classes on the buttons to make it clearer when options are checked or unchecked
