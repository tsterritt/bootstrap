Range

Note: range elements don't have a way to display the selected value. In these examples, a small java script snippets displays the value based on the selection in the range.

Default

By default the range has a range from 0 to 100 with 100 steps. form-label class adds formatting to the label, and the form-range class on the input element adds styling to the input.

Max

Adding min max to the input element adds a number of steps to the slider and sets the values accordingly. Use value attribute to spefic the default value.

Increments

Adding step attribute to the range element to specify the number of steps
