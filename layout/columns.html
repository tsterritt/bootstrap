Vertical Alignment

Use align-items to vertically align columns in the row. Use align-self to align the columns individually. The row must have a specified height

Column Wrapping

If the specified column breakpoints in a row total greater than 12, the row will automatically wrap.

.col-8
.col-5
.col-4
