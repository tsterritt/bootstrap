Horizontal Gutters

gx|y-* classes on the row add width between column content. x=horizontal, y=vertical. Note: gx|y classes split the width of the gutter - 1/2 on one side, 1/2 on the other. There are responsive classes (gx-lg-5 e.g.)

Bottom-line - similar to adding padding but saves typing since gx-* classes go on the row not the columns. Messes up row widths (rows 3 and 4).

* Row1 is for comparison - just default row and column classes, with no padding or gutters. Except the default, which looks like it's px-3

* Row2 adds a px-3 to each column, which increases the space between the columns

* Row3 achieves a similar result to row2 using a gutter (gx-3) on the row instead of a px-3 on each column, which saves typing.

* Row4 uses gx-0 on the row, which overrides the default padding that bootstrap adds.
