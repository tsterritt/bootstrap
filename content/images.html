Non Responsive Image

Responsive Image

img-fluid class makes image responsive. The image takes up the real estate of the parent and scales for the viewport. The two 1200x600 images in the grid scale to take up the full real estate available in their respective columns, but scale responsively

Rounded corners

rounded- classes control corner rounding.

.rounded-top
.rounded-bottom
.rounded-end
.rounded-start
.rounded-circle
.rounded-pill

Thumbnail

img-thumbnail class adds a border, responsiveness, and rounds the corners. Note - thumbnail doesn't resize the image.

.img-thumbnail
.img-thumbnail .rounded-0
.img-thumbnail rounded-pill

Float

use float-* classes to position elements. Note, if floated elements are taller than the container, it will overflow outside of the container. The clearfix class fixes this.

No clearfix class, floated elements extend outside of containing div
Using clearfix adjusts containing div to contain the floated elements

When floating images, subsequent elements will wrap around the image. Use clear class to 'clear' the float from elements to the left, right, or both.

Spacing utilities

d-block and mx-* classes to add margins that align images. D-block causes an element to take up a whole line, and m*-auto adds margins to the start (ms-aut), end (me-auto), or on both sides (mx-auto)

Image Caption

Using figure element and figure-img class around an image allows the image to have a caption with padding between the image and the caption. The figcaption element with added figure-caption class style the caption. The caption text can be position with the text-center, text-end, and text-start (the default) classes
