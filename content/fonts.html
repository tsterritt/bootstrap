Inline Text

Inline tags can have the same visual result, but have different semantic purposes

<mark> highlights text

<del> adds strikethrough, for deleted text

<s> also adds strikethrough, specific for strikethrough

<ins> inserted text, adds underline

<u> also adds underline, but for underlining text

<small> reduces font size

<strong> bolds text

<b> also bolds text

<em> italicizes text

<i> also italicizes, but for technical terms or alternate voice
